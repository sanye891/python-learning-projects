# 打工人能量条

一个基于 PyQt5 的桌面应用程序，用于显示工作时间进度和下班倒计时。

## 功能特点

- 实时显示当前时间
- 可视化展示工作日进度
- 倒计时显示距离下班的剩余时间
- 可自定义下班时间
- 无边框窗口设计，支持拖拽移动
- 包含励志语录展示
- 下班时间提醒功能
- 下班时间记录功能

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 使用说明

1. 程序启动后会显示当前时间、工作进度条和下班倒计时
2. 点击"设置"按钮可以修改下班时间（默认为18:00）
3. 点击"重置"按钮可以重置下班提醒状态
4. 点击"最小化"按钮可以最小化窗口
5. 可以通过标题栏区域拖拽移动窗口
6. 到达下班时间时会弹出提醒窗口

## 项目结构

- [main.py] - 程序入口文件
- [energy_bar.py] - 主窗口界面和逻辑
- [settings.py] - 设置窗口
- [data_manager.py] - 数据管理
- [cat.png] - 程序图标

## 系统要求

- Python 3.6+
